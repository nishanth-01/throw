# Throw
A simple 2D target hitting game inspired from golf

> In development, yet to be deployed

> All outputs are in the `console` for now

> Improvements on the way

## Controls
| Key | Action |
|-|-|
| `'w' | 'W'` | Increase angle |
| `'s' | 'S'` | Decrease angle |
| `'a' | 'A'` | Increase force |
| `'d' | 'D'` | Decrease force |
| `' '` | Launch |

## Run
```bash
npm run start
```

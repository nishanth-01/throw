# Throw
A simple 2D target hitting game inspired from golf

## Controls
| Key | Action |
|-|-|
| `'w' \| 'W'` | Increase angle |
| `'s' \| 'S'` | Decrease angle |
| `'a' \| 'A'` | Increase force |
| `'d' \| 'D'` | Decrease force |
| `' '` | Launch |

## Run
### Linux
```bash
git clone https://github.com/nishanth-01/throw.git
cd throw
npm run start

```
